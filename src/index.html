World Clock Wallpaper

Version <%= version %>

A customizable Chrome extension that displays multiple digital world clocks as your new tab wallpaper.

Features:

  • Main central clock with your local timezone
  • Add multiple clocks from different world regions
  • Customizable colors, fonts, and sizes
  • 12/24 hour time formats
  • Multiple date format options
  • Responsive design for all devices
  • Persistent settings storage

Technical Details:

Built with vanilla JavaScript, HTML5, and CSS3

Uses Chrome Extension Manifest V3

Powered by Intl.DateTimeFormat API for timezone support

Support:

For issues or suggestions, please visit the extension repository.

Background Color

Changes the background color of the entire new tab page. Choose a color that's comfortable for your eyes and complements the clock display.

Font Color

Sets the text color for all clock displays and interface elements. Ensure good contrast with your background color.

üí° Tip: White text on dark backgrounds or black text on light backgrounds provides best readability.
Font Family

Choose from different font styles for the clock display. Each font has its own personality and readability characteristics.

  • Arial/Helvetica: Clean, modern sans-serif fonts
  • Georgia/Times: Classic serif fonts
  • Courier New: Monospace font for digital clock feel
  • Verdana: Wide, readable sans-serif font
Font Size

Adjusts the size of all text elements. Range from 12px (small) to 48px (very large).

üí° Tip: Larger sizes are great for viewing from a distance or on high-resolution displays.

‚è∞ Time & Date Settings

Time Format

Choose how time is displayed:

  • 24-hour: 00:00:00 to 23:59:59 (e.g., 14:30:00)
  • 12-hour: 12:00:00 AM/PM to 11:59:59 PM (e.g., 2:30:00 PM)
Date Format

Select how the date appears in the clock display:

  • Long: Monday, January 1, 2024
  • Medium: Jan 1, 2024
  • Short: 1/1/2024

üíæ Configuration Management

Saving Settings

Click the "Save Configuration" button to store your preferences. Your settings are saved to Chrome's sync storage and will be available across all your signed-in devices.

Reset to Default

Click "Reset to Default" to restore all settings to their original values:

  • Background: Dark blue (#1a1a2e)
  • Text: White (#ffffff)
  • Font: Arial, 16px
  • Time: 24-hour format
  • Date: Long format
  • No additional timezones

üîß Troubleshooting

Clocks Not Updating

If clocks stop updating:

  • Refresh the new tab page (Ctrl+R or Cmd+R)
  • Check if Chrome extensions are enabled
  • Try disabling other new tab extensions
Settings Not Saving

If your configuration doesn't persist:

  • Ensure you're signed into Chrome
  • Check if Chrome sync is enabled
  • Try clicking Save again after a few seconds
