World Clock Wallpaper

Version 1.0

A customizable Chrome extension that displays multiple digital world clocks as your new tab wallpaper.

Features:

  • Main central clock with your local timezone
  • Add multiple clocks from different world regions
  • Customizable colors, fonts, and sizes
  • 12/24 hour time formats
  • Multiple date format options
  • Responsive design for all devices
  • Persistent settings storage

Technical Details:

Built with vanilla JavaScript, HTML5, and CSS3

Uses Chrome Extension Manifest V3

Powered by Intl.DateTimeFormat API for timezone support
